Cara Kompres PDF untuk Dikirim via Email (Gmail & Outlook) 2026

Gmail membatasi lampiran hingga 25MB, Outlook hingga 20MB. Jika PDF Anda melebihi batas ini, panduan ini akan membantu Anda mengompresnya secara gratis agar bisa dikirim via email.

Batas Ukuran Lampiran Email

  • Gmail: 25MB
  • Outlook: 20MB
  • Yahoo Mail: 25MB
  • Email Perusahaan: biasanya 10-20MB
Perhatian

Meski PDF Anda di bawah 25MB, server email penerima mungkin menolak file yang terlalu besar. Disarankan mengompres PDF di bawah 10MB sebelum mengirim email.

Langkah Kompres PDF untuk Email

1

Buka ShrinkPDF

Akses ShrinkPDF di browser, tanpa perlu daftar atau install apapun.

2

Upload PDF Anda

Klik pilih file atau seret dan lepas PDF ke area upload.

3

Pilih Mode Kompresi

Untuk email, mode "Seimbang" biasanya sudah cukup. Jika masih terlalu besar, gunakan "Maksimum".

4

Unduh dan Kirim sebagai Lampiran

Unduh PDF yang sudah dikompres, lalu lampirkan ke email seperti biasa.

Jika Masih Terlalu Besar?

  • Kompres ulang dengan mode Maksimum
  • Pisahkan PDF besar menjadi beberapa email
  • Gunakan Google Drive atau OneDrive, kirim link berbagi sebagai gantinya

Pertanyaan yang Sering Ditanyakan

Apakah penerima akan melihat penurunan kualitas? ▼
Dengan mode Seimbang, kualitas teks dan gambar hampir tidak berubah. Mode Maksimum mungkin sedikit mengurangi ketajaman gambar, tapi untuk dokumen bisnis biasanya masih sangat layak.
Apakah isi PDF saya aman? ▼
Sepenuhnya aman. ShrinkPDF memproses file secara lokal di browser Anda. File tidak pernah diunggah ke server manapun.
Bisakah mengompres PDF yang dilindungi kata sandi? ▼
Jika PDF memiliki proteksi kata sandi, hapus proteksinya terlebih dahulu, lalu kompres dengan ShrinkPDF.
