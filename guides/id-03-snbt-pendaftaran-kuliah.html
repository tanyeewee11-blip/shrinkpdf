Cara Kompres PDF untuk Pendaftaran SNBT/SNBP dan PTN (2026)

Sedang daftar SNBT, SNBP, atau jalur mandiri PTN dan sistem menolak unggahan KK, KTP, rapor, atau dokumen lain karena "ukuran file terlalu besar"? Ini masalah yang sangat umum terjadi setiap musim pendaftaran. Berikut cara memperkecil ukuran dokumen kamu dalam waktu kurang dari satu menit, gratis.

Batas Ukuran File di Portal Pendaftaran

Portal pendaftaran SNBT, SNBP, dan PTN jalur mandiri biasanya menetapkan batas ukuran yang cukup ketat untuk setiap dokumen, sering kali antara 200KB sampai 2MB per file. Ini termasuk:

  • Pas foto formal — biasanya dibatasi dalam ukuran KB, bukan MB
  • Kartu Keluarga (KK) dan KTP orang tua — sering hasil scan atau foto HP yang ukurannya besar
  • Rapor atau transkrip nilai — dokumen multi halaman yang mudah membesar
  • Surat keterangan tidak mampu (untuk jalur bantuan biaya) — biasanya hasil scan dari surat fisik
  • Sertifikat prestasi atau portofolio — untuk jalur prestasi atau seni

Dokumen yang difoto langsung dengan kamera HP pada resolusi penuh sering melebihi batas ini jauh, satu foto dari kamera HP modern bisa mencapai 3 sampai 8MB sebelum dijadikan PDF.

Dokumen yang Sering Ditolak

Penyebab Umum File Ditolak
  • Foto dari kamera HP langsung dijadikan PDF — resolusi kamera HP modern jauh lebih tinggi dari yang dibutuhkan untuk dokumen administratif.
  • Beberapa dokumen digabung dalam satu file — misalnya KK, KTP ayah, dan KTP ibu dalam satu PDF bisa melebihi batas walaupun masing-masing kecil.
  • Rapor hasil scan dengan banyak halaman berwarna — rapor dari kelas 10 sampai 12 bisa puluhan halaman, dan jika di-scan berwarna ukurannya membesar drastis.
  • PDF dari aplikasi scan yang tidak otomatis kompres — beberapa aplikasi scan pihak ketiga menghasilkan PDF yang lebih besar dari seharusnya.

Cara Kompres Dokumen untuk Pendaftaran

1

Buka ShrinkPDF di HP atau komputer

Kunjungi ShrinkPDF.fyi, bisa di Chrome, Safari, atau browser lain. Tidak perlu instal aplikasi, bisa langsung dipakai saat kamu sedang mengisi formulir pendaftaran.

2

Unggah file yang ditolak

Klik "Pilih File PDF" dan pilih dokumen yang gagal diunggah. Kalau dokumen kamu masih berupa foto (JPG/PNG), gunakan alat Images to PDF dulu untuk mengubahnya ke PDF.

3

Pilih Maksimum dan tunggu prosesnya

Pilih tingkat Maksimum untuk pengurangan ukuran terbesar, biasanya 70 sampai 85 persen. Prosesnya cuma beberapa detik.

4

Unduh dan unggah ulang

Unduh file hasil kompresi, kembali ke portal pendaftaran, dan coba unggah lagi. File yang awalnya 4 sampai 5MB biasanya turun jadi di bawah 1MB.

🔒 Dokumen Identitas Kamu Aman

ShrinkPDF memproses semua file di browser kamu, KK, KTP, akta kelahiran, dan rapor kamu tidak dikirim ke server mana pun. Semua proses kompresi berjalan langsung di HP atau komputer kamu.

Bila Unggahan Masih Gagal

Kalau file sudah di bawah batas ukuran tapi portal masih menolak, coba langkah berikut:

  • Pastikan format file sesuai — beberapa bagian formulir hanya menerima PDF, ada juga yang hanya menerima JPG. Baca instruksi di setiap kolom unggahan dengan teliti.
  • Pisahkan dokumen yang digabung — kalau kamu menggabungkan beberapa dokumen dalam satu PDF, gunakan alat Split untuk mengunggah setiap dokumen secara terpisah sesuai kategori yang diminta.
  • Coba browser atau perangkat lain — beberapa masalah unggahan terkait dengan browser tertentu, terutama Safari versi lama di iPhone.
  • Periksa koneksi internet — unggah file di jaringan data yang lambat kadang timeout sebelum selesai walaupun ukuran file sudah sesuai.
  • Lapor ke panitia pendaftaran — kalau masalah terus terjadi, hubungi help desk SNBT/SNBP atau panitia PTN terkait, kadang ada kendala sistem di luar kontrol kamu.

Kompres Dokumen Pendaftaranmu Sekarang — Gratis

Tanpa daftar. File kamu tidak meninggalkan perangkat kamu.

Pertanyaan yang Sering Diajukan

Berapa batas ukuran file untuk pendaftaran SNBT atau SNBP? ▼
Batas ukuran file untuk dokumen pendaftaran SNBT dan SNBP biasanya berkisar 200KB sampai 2MB per dokumen, tergantung jenis dokumen seperti pas foto, KK, atau rapor. Batas ini bisa berubah setiap tahun, jadi selalu periksa petunjuk teknis resmi yang dirilis sebelum periode pendaftaran dibuka.
Kenapa scan KK atau KTP saya selalu besar ukurannya? ▼
Foto dari kamera HP biasanya beresolusi tinggi (12MP atau lebih), menghasilkan file beberapa MB sebelum dijadikan PDF. Kalau beberapa dokumen digabung jadi satu file atau di-scan berwarna, ukurannya akan jauh lebih besar dari yang dibutuhkan. Kompres dengan ShrinkPDF biasanya menurunkan ukuran 70 persen atau lebih tanpa mengurangi keterbacaan.
Apakah aman mengompres dokumen seperti KTP dan KK? ▼
Ya. ShrinkPDF memproses semua file di browser kamu sendiri menggunakan JavaScript. Dokumen identitas seperti KTP, KK, dan akta kelahiran tidak pernah dikirim ke server mana pun selama proses kompresi, semuanya terjadi langsung di perangkat kamu.
Rapor saya banyak halaman, apakah perlu dikompres satu per satu? ▼
Tidak perlu. Kamu bisa unggah seluruh rapor sebagai satu file PDF ke ShrinkPDF dan kompres semuanya sekaligus. Kalau hasilnya masih terlalu besar, gunakan alat Split untuk membagi rapor per semester atau per beberapa halaman, lalu kompres masing-masing bagian.
Bisa kompres dokumen lewat HP saat mengisi formulir pendaftaran? ▼
Ya. ShrinkPDF berfungsi penuh di browser HP, Chrome dan Samsung Internet di Android, Safari di iPhone. Buka shrinkpdf.fyi di tab baru, kompres dokumen yang ditolak, lalu kembali ke tab formulir pendaftaran untuk mengunggah file yang sudah lebih kecil.
