Cara Kompres PDF untuk Pendaftaran Beasiswa LPDP (2026)

Sedang mengisi formulir pendaftaran LPDP dan sistem menolak unggahan transkrip, ijazah, atau sertifikat karena ukuran file terlalu besar? Batas Ukuran File pada Aplikasi LPDP

Sistem pendaftaran beasiswa LPDP biasanya menetapkan batas ukuran untuk setiap dokumen yang diunggah, baik itu transkrip akademik, ijazah, sertifikat bahasa (TOEFL/IELTS), maupun surat rekomendasi. Batas ini bisa berbeda setiap tahun dan setiap jenis dokumen, jadi selalu cek instruksi resmi di portal pendaftaran sebelum mengunggah.

Hal yang Perlu Diperhatikan
  • Batas ukuran biasanya diterapkan per dokumen, bukan untuk total keseluruhan
  • Format yang umum diterima: PDF (paling disarankan), kadang JPG untuk dokumen tertentu
  • Batas bisa berubah setiap periode pendaftaran, periksa selalu petunjuk teknis terbaru

Dokumen yang Sering Terlalu Besar

Berdasarkan pengalaman pendaftar di periode-periode sebelumnya, dokumen berikut paling sering melebihi batas ukuran:

  • Transkrip nilai hasil scan berwarna pada 300 DPI — biasanya 3 sampai 6MB untuk transkrip beberapa halaman. Setelah kompresi maksimum, bisa turun ke bawah 1MB.
  • Ijazah dengan stempel dan tanda tangan berwarna — stempel basah dan cap berwarna sering membuat ukuran file membesar walaupun teksnya sedikit.
  • Sertifikat TOEFL/IELTS dalam format scan resolusi tinggi — sertifikat resmi sering di-scan dengan kualitas tinggi karena ada watermark atau hologram, padahal untuk upload cukup resolusi standar.
  • Surat rekomendasi yang di-scan dari hasil print berwarna — surat dengan kop surat dan logo institusi berwarna bisa membesarkan ukuran file secara signifikan.

Cara Kompres Dokumen Pendaftaran

1

Cek ukuran file terlebih dahulu

Di Windows, klik kanan file lalu pilih Properties dan lihat "Size". Di Mac, klik kanan lalu Get Info. Di HP, tekan lama file di aplikasi Files atau Google Files.

2

Buka ShrinkPDF

Kunjungi ShrinkPDF.fyi di browser. Berfungsi di Windows, Mac, Android, dan iPhone, tanpa instal aplikasi tambahan.

3

Unggah dan pilih Maksimum

Klik "Pilih File PDF" atau seret file kamu. Pilih tingkat Maksimum, ini mengurangi ukuran 70 sampai 85 persen sambil menjaga teks tetap jelas terbaca, cocok untuk batas ukuran yang ketat pada formulir LPDP.

4

Unduh dan pastikan sesuai batas

Setelah diunduh, periksa kembali ukuran file. Pastikan jelas di bawah batas yang ditentukan, usahakan ada margin aman supaya tidak mepet.

5

Buka dan periksa isi dokumen

Sebelum mengunggah ke aplikasi LPDP, buka file hasil kompresi dan pastikan nilai, nama, nomor ijazah, dan stempel masih terlihat jelas.

6

Unggah ke sistem pendaftaran

Lengkapi pendaftaran kamu dengan dokumen yang sudah dikompres. Jika sistem masih menolak, coba browser lain atau refresh sesi pendaftaran.

🔒 Dokumen Akademik Kamu Aman

ShrinkPDF memproses semua file langsung di browser kamu. Transkrip, ijazah, dan sertifikat kamu tidak pernah dikirim ke server mana pun. Ini berbeda dengan banyak tool kompres online yang mengunggah file kamu ke cloud server mereka untuk diproses, perbedaan ini penting untuk dokumen akademik dan identitas yang bersifat pribadi.

Tips Tambahan untuk Pendaftar LPDP

  • Siapkan semua dokumen jauh sebelum deadline — proses scan, kompres, dan cek ulang butuh waktu, terutama kalau kamu harus scan ulang dokumen fisik.
  • Beri nama file yang jelas dan konsisten — misalnya "Transkrip_NamaLengkap.pdf", supaya mudah dikenali dan tidak tertukar saat upload banyak dokumen sekaligus.
  • Kalau dokumen terdiri dari banyak halaman terpisah, gabungkan dulu menggunakan alat Merge, baru kompres keseluruhan file sekali, ini kadang menghasilkan ukuran total yang lebih kecil dibanding file terpisah.
  • Simpan salinan asli sebelum dikompres — meskipun ShrinkPDF membuat file baru dan tidak mengubah file asli, tetap simpan salinan dokumen asli di folder lain untuk berjaga-jaga.

Kompres Dokumen Pendaftaranmu Sekarang — Gratis

Tanpa daftar. File kamu tidak meninggalkan perangkat kamu.

Pertanyaan yang Sering Diajukan

Berapa batas ukuran file untuk unggahan dokumen LPDP? ▼
Batas ukuran file untuk dokumen pendukung LPDP bisa berbeda tergantung jenis dokumen dan tahun pendaftaran, biasanya berkisar 1 sampai 5MB per dokumen. Selalu periksa instruksi resmi di aplikasi pendaftaran LPDP untuk batas yang berlaku pada periode kamu mendaftar.
Apakah aman mengompres dokumen akademik seperti transkrip dan ijazah? ▼
Ya. ShrinkPDF memproses semua file di browser kamu sendiri menggunakan JavaScript. Transkrip, ijazah, sertifikat, dan dokumen identitas kamu tidak pernah dikirim ke server mana pun selama proses kompresi, semuanya terjadi langsung di perangkat kamu.
Transkrip saya masih lebih besar dari batas setelah dikompres, apa yang harus dilakukan? ▼
Coba scan ulang dokumen dalam hitam putih pada 150 DPI, ini biasanya jauh lebih efektif dibanding kompresi ulang dari file yang sudah dikompres. Jika transkrip terdiri dari banyak halaman, gunakan alat Split untuk memisahkannya jika sistem pendaftaran mengizinkan unggahan multi file.
Apakah saya bisa menggunakan format JPG untuk sertifikat? ▼
Sebagian sistem pendaftaran menerima JPG untuk dokumen seperti foto atau sertifikat singkat, tapi PDF lebih disarankan karena bisa menampung beberapa halaman dalam satu file dan terlihat lebih rapi. Jika dokumen kamu awalnya berupa foto, kamu bisa pakai alat Images to PDF untuk mengubahnya ke PDF lebih dulu.
Bisa kompres dokumen pendaftaran lewat HP saat mengisi formulir? ▼
Ya. ShrinkPDF berfungsi penuh di browser HP, Chrome dan Samsung Internet di Android, Safari di iPhone. Buka shrinkpdf.fyi, pilih file dari aplikasi File atau Google Drive, kompres, lalu unduh file yang siap diunggah ke sistem pendaftaran LPDP.
