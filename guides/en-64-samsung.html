Method 1: Using Chrome (Recommended)

Chrome is pre-installed on most Samsung Galaxy phones and is the most reliable browser for ShrinkPDF.

1

Open Chrome and go to shrinkpdf.fyi

Tap the Chrome app on your Samsung. In the address bar, type shrinkpdf.fyi and tap Go.

2

Tap "Choose PDF File"

Tap the upload button. Samsung's file picker opens. Your PDF might be in My Files → Downloads, or in a cloud storage app like Google Drive or OneDrive if you downloaded it from there.

3

Select your PDF

Navigate to the file and tap it. The file name and size appear on screen. If your PDF is in Google Drive and you don't see it in "My Files", tap the three-line menu in the file picker and select "Browse" → "Google Drive".

4

Choose compression level and compress

Select Maximum for portal uploads (LHDN, JPA, UPU, MyEG) or Balanced for email and sharing. Tap Compress PDF Now. The compression runs on your Samsung's processor — no internet connection needed for the compression itself.

5

Download the compressed file

When compression finishes, the before/after sizes appear. Tap Download Compressed PDF. Chrome shows a download notification at the top of the screen. The file saves to My Files → Downloads.

Method 2: Using Samsung Internet Browser

Samsung Internet (the blue compass icon) is Samsung's built-in browser and works equally well.

1

Open Samsung Internet and go to shrinkpdf.fyi

Tap the Samsung Internet app (blue globe/compass icon). Type shrinkpdf.fyi in the address bar.

2

Tap the upload area or "Choose PDF File"

Samsung Internet's file picker may look slightly different from Chrome's — it opens Samsung's native file manager directly. Navigate to Downloads or wherever your PDF is saved.

3

Select level and compress

Same as the Chrome method. Select your compression level and tap Compress.

4

Download

Tap Download. Samsung Internet shows a download progress bar at the bottom. The file saves to Downloads.

Where to Find the Compressed File on Samsung

After downloading, your compressed PDF is in:

  • My Files → Downloads (the main Samsung file manager)
  • The file is named yourfilename_shrinkpdf.pdf
  • You can also access recent downloads by tapping the Chrome menu (three dots) → Downloads
Tip: Pin "My Files" to your home screen

If you frequently compress and upload PDFs, add "My Files" to your Samsung home screen for quick access. Long-press the My Files icon in the app drawer → "Add to Home Screen".

How to Share or Upload After Compressing

  • WhatsApp: In the WhatsApp conversation, tap the 📎 icon → Document → navigate to Downloads → select the compressed PDF. Do not use "Gallery" — that converts the PDF to images.
  • Email: Open Gmail or your email app. Start a new email → Attach file → Downloads → select the file.
  • Portal upload: In the portal's browser tab (LHDN, UPU, MyEG), when the upload field appears, tap "Browse" or the file icon → navigate to Downloads → select the compressed PDF.
  • Google Drive: Open Google Drive → tap + → Upload → My Files → Downloads → select the file.

Troubleshooting

The file picker doesn't show my PDF ▼
Change the file type filter in the picker from "Recent" to "All files" or navigate manually to My Files → Downloads. If the PDF was downloaded from a browser, it will be in Downloads. If it came from WhatsApp, check My Files → Internal Storage → WhatsApp → Media → WhatsApp Documents.
Compression is very slow on my Samsung ▼
Compression speed depends on your Samsung model and file size. Budget models (Galaxy A series) may take 30–60 seconds for a 10MB PDF. Mid-range and flagship models (Galaxy S series) are significantly faster. Keep the browser in the foreground — switching apps may pause the process on some Samsung One UI versions.
The download failed or the file is corrupted ▼
This occasionally happens if Samsung's battery optimisation paused the browser mid-download. Go to Settings → Apps → Chrome (or Samsung Internet) → Battery → set to "Unrestricted". Then try the compression again. Alternatively, ensure you have at least 100MB of free storage on your Samsung.
Can I compress PDF on Samsung without internet? ▼
You need an internet connection to load the ShrinkPDF page initially. Once the page has loaded, the actual compression runs locally on your Samsung — it does not send your file over the internet. If you lose connection during compression, the compression will still complete as long as the page remains loaded.
