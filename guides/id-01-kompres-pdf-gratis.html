Cara Kompres PDF Gratis — Tanpa Upload ke Server (2026)

Kenapa Ukuran File PDF Bisa Besar?

Banyak orang bingung kenapa file PDF yang isinya cuma beberapa halaman bisa mencapai 5MB atau lebih. Ini beberapa penyebab utamanya:

  • Pengaturan scanner terlalu tinggi: Sebagian besar aplikasi scan seperti CamScanner atau Adobe Scan menggunakan 300 DPI sebagai default. Ini bagus untuk hasil cetak, tapi untuk upload dokumen, 150 DPI sudah lebih dari cukup dan ukurannya bisa 75 persen lebih kecil.
  • Hasil scan berwarna: Dokumen seperti slip gaji, transkrip, atau surat keterangan biasanya tidak perlu warna. Scan hitam putih bisa menghasilkan file 60 sampai 70 persen lebih kecil.
  • Banyak halaman: Rekening koran 6 bulan yang di-scan bisa mencapai 15MB. Setiap halaman menambah 1 sampai 3MB di pengaturan default.
  • Gambar di dalam PDF: PDF yang berisi foto atau logo beresolusi tinggi akan jadi sangat besar meskipun isi teksnya sedikit.

Cara Kompres PDF (Langkah demi Langkah)

1

Buka ShrinkPDF di browser

Kunjungi ShrinkPDF.fyi menggunakan Chrome, Safari, atau Firefox. Bisa dipakai di komputer, HP Android, atau iPhone. Tidak perlu instal aplikasi, tidak perlu daftar akun.

2

Unggah file PDF kamu

Klik tombol "Pilih File PDF" atau seret file langsung ke kotak unggah. Tidak ada batas ukuran file, jadi kamu bisa kompres PDF seberapa besar pun.

3

Pilih tingkat kompresi

Untuk pendaftaran beasiswa, kampus, atau formulir online lainnya, pilih Maksimum. Untuk dokumen yang butuh kualitas gambar lebih baik, pilih Seimbang.

4

Tunggu proses kompresi selesai

Biasanya hanya 5 sampai 30 detik tergantung ukuran file. Semua proses terjadi di browser kamu, file tidak dikirim ke internet.

5

Unduh dan periksa hasilnya

Klik tombol unduh. Sebelum dikirim, buka file hasil kompresi dan pastikan semua teks masih terbaca jelas, terutama nomor identitas, nominal, dan tanda tangan.

🔒 Privasi Kamu Terjaga

ShrinkPDF memproses semua file langsung di browser kamu menggunakan JavaScript. KTP, slip gaji, rekening koran, semua tidak pernah meninggalkan perangkat kamu dan tidak dikirim ke server mana pun. Kamu bisa cek sendiri, buka tab Network di browser saat kompresi berjalan, tidak akan ada pengiriman file yang terlihat.

Pilih Tingkat Kompresi yang Tepat

Panduan Tingkat Kompresi
  • Maksimum — Ukuran paling kecil (pengurangan 70 sampai 85 persen). Cocok untuk: formulir pendaftaran online, kirim lewat WhatsApp, email dengan batas ukuran kecil. Teks tetap tajam, gambar mungkin sedikit kurang detail saat di-zoom.
  • Seimbang — Seimbang antara kualitas dan ukuran (pengurangan 50 sampai 65 persen). Cocok untuk: dokumen yang akan dicetak ulang, portofolio, atau presentasi.
  • Ringan — Pengurangan kecil (20 sampai 35 persen). Cocok untuk: file yang hampir memenuhi batas ukuran dan kamu ingin kualitas asli tetap maksimal.

Kompres ke Ukuran Tertentu (1MB, 500KB, 200KB)

Banyak sistem pendaftaran online menetapkan batas ukuran yang spesifik, bukan cuma "sekecil mungkin". Berikut panduan kasar target ukuran dan pengaturan yang biasanya cocok:

  • Target 1MB — Untuk PDF sekitar 2 sampai 5 halaman hasil scan, pilih tingkat kompresi Seimbang. Untuk dokumen lebih banyak halaman atau hasil scan berwarna, gunakan Maksimum.
  • Target 500KB — Hampir selalu butuh tingkat Maksimum, kecuali dokumennya cuma 1 halaman teks tanpa gambar.
  • Target 200KB atau lebih kecil — Ini batas yang cukup ketat. Kalau tingkat Maksimum masih belum cukup, coba scan ulang dokumen dalam hitam putih di 100 sampai 150 DPI (lihat tips di bawah), baru kompres hasil scan barunya.

ShrinkPDF tidak punya kolom "masukkan ukuran target" karena hasil akhirnya sangat tergantung isi dokumen — jumlah halaman, ada gambar atau tidak, warna atau hitam putih — bukan cuma pengaturan kompresi saja. Cara paling akurat adalah coba tingkat Seimbang dulu, cek ukurannya, baru naikkan ke Maksimum kalau masih terlalu besar.

Tips Sebelum Scan Dokumen

Kalau kamu belum scan dokumennya, pengaturan yang tepat dari awal bisa menghindari masalah ukuran file sama sekali:

  • Gunakan 150 DPI, bukan 300 DPI — 150 DPI sudah cukup untuk hampir semua formulir online. Teks tetap tajam di layar. Di CamScanner: Pengaturan → Kualitas Scan → Standard.
  • Scan dalam hitam putih — Kebanyakan dokumen resmi (slip gaji, transkrip, surat keterangan) tidak perlu warna. Hitam putih mengurangi ukuran file 60 sampai 70 persen.
  • Gunakan Microsoft Lens (gratis) — Aplikasi ini menghasilkan PDF lebih kecil dibanding CamScanner secara default dan deteksi tepi otomatisnya lebih baik. Tersedia di Android dan iPhone.
  • Scan halaman satu per satu — Kalau sistem pendaftaran mengizinkan upload beberapa file, scan setiap dokumen secara terpisah agar lebih mudah diatur.

Masih Terlalu Besar Setelah Dikompres?

Kalau file masih melebihi batas setelah kompresi maksimum, coba langkah berikut:

  • Scan ulang dengan pengaturan lebih rendah: Kalau kamu masih punya dokumen fisik aslinya, scan ulang di 100 sampai 150 DPI hitam putih. Satu halaman A4 di 150 DPI hitam putih biasanya hanya 50 sampai 150KB.
  • Pisahkan dokumen: Gunakan alat Split ShrinkPDF untuk membagi PDF jadi beberapa file lebih kecil. Unggah setiap bagian secara terpisah kalau sistem mengizinkan.
  • Hapus halaman yang tidak perlu: Gunakan alat Reorder untuk menghapus halaman kosong atau duplikat sebelum dikompres.
  • Hubungi panitia atau admin sistem: Beberapa sistem punya cara pengiriman alternatif lewat email untuk dokumen yang memang besar.

Kompres PDF Kamu Sekarang — Gratis

Tanpa daftar. Tanpa batas ukuran. File kamu tidak meninggalkan perangkat kamu.

Pertanyaan yang Sering Diajukan

Apakah aman mengompres dokumen pribadi seperti KTP atau slip gaji? ▼
Ya, sepenuhnya aman. ShrinkPDF memproses semua file langsung di browser kamu. KTP, slip gaji, rekening koran, dan dokumen pajak kamu tidak pernah dikirim ke server mana pun. Kamu bisa putuskan koneksi internet setelah halaman terbuka dan kompresi tetap berjalan, ini buktinya tidak ada proses yang terjadi di luar perangkat kamu.
Berapa lama proses kompres PDF? ▼
Biasanya 5 sampai 30 detik untuk file normal (di bawah 20MB). File yang lebih besar atau komputer yang lebih lama mungkin butuh waktu 1 sampai 2 menit. Proses ini tergantung kekuatan prosesor perangkat kamu, bukan kecepatan internet.
Bisa kompres PDF lewat HP? ▼
Ya. ShrinkPDF berfungsi penuh di browser HP, Chrome dan Samsung Internet di Android, Safari di iPhone. Buka shrinkpdf.fyi di browser, ketuk "Pilih File PDF", pilih file dari aplikasi File atau Google Drive, kompres, lalu unduh. Tidak perlu instal aplikasi apa pun.
Apakah kualitas PDF akan menurun setelah dikompres? ▼
Teks tetap tajam dan mudah dibaca di semua tingkat kompresi. Hanya kualitas gambar (foto, logo) yang mungkin sedikit berkurang di tingkat Maksimum. Untuk dokumen resmi seperti slip gaji, transkrip, dan surat, hasil kompresi terlihat hampir sama dengan file aslinya. Selalu buka dan periksa file hasil kompresi sebelum dikirim.
PDF saya masih ditolak sistem walaupun ukurannya sudah di bawah batas. Kenapa? ▼
Terkadang sistem pendaftaran mengalami masalah teknis yang tidak terkait ukuran file. Coba refresh halaman dan unggah ulang. Kalau masih gagal, coba browser yang berbeda. Kalau masalah terus berlanjut, hubungi bagian bantuan teknis sistem tersebut, kadang batas ukuran sebenarnya berbeda dari yang tertulis.
