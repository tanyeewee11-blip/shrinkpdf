General Tips for Hitting Any Size Limit

  • When in doubt, target 2MB. Almost every platform that has a limit accepts 2MB. If you compress to 2MB, you'll clear 95% of upload portals you'll ever encounter.
  • Start with Maximum compression. For documents being submitted to portals (not printed), Maximum compression is visually indistinguishable at normal screen viewing. Start there and go up in quality only if rejected.
  • Split large documents. If a multi-page document still exceeds the limit after Maximum compression, use the Split PDF tool to extract only the pages being requested and compress those.
  • Ask before assuming. Some portals list a limit but actually accept larger files. Others are strict. If the portal rejects a 2MB file, check if there's a different format requirement (e.g., JPG only).

FAQ

What's the safest PDF size to target for any upload? ▼
2MB. Almost every platform that has a file size limit accepts 2MB. Government portals (LHDN, UPU, MyEG), job portals, university applications, and most email providers all accept 2MB files. If you compress to 2MB, you will clear 95% of upload scenarios without needing to check the specific limit.
Why do different platforms have such different limits? ▼
Email limits are set by mail server administrators and reflect infrastructure costs and spam prevention. Portal limits are often set by government IT policy, which tends to be conservative. Messaging apps have high limits because they're designed for media sharing. Cloud storage has essentially no limits because storage is their core business. The practical result is that you need different strategies for different platforms.
Can I always compress a PDF to under 2MB? ▼
Almost always. The exception is a PDF that is already highly compressed — for example, a PDF that was already compressed by a scanner app before you received it. In that case, ShrinkPDF may only achieve 10–20% reduction because there's little left to compress. If you hit this scenario, try splitting the document and only submitting the required pages, or re-scan the original document at lower DPI settings.
