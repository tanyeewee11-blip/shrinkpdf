Cara Gabungkan PDF Gratis — Tanpa Upload ke Server

Punya beberapa file PDF terpisah yang perlu jadi satu dokumen — untuk lamaran kerja, berkas beasiswa, atau pengajuan pinjaman? Anda bisa menggabungkannya dalam waktu kurang dari satu menit, gratis, tanpa instal apa pun, dan file Anda tidak pernah dikirim ke server mana pun.

Kenapa Perlu Menggabungkan PDF?

Beasiswa dan pendaftaran kuliah

Pendaftaran beasiswa seperti LPDP atau jalur seleksi kampus seperti SNBT sering meminta beberapa dokumen sekaligus — transkrip nilai, surat rekomendasi, esai, dan KTP — digabung jadi satu file PDF karena sistem hanya menyediakan satu kolom unggahan.

Lamaran kerja

Gabungkan CV, surat lamaran, dan portofolio jadi satu PDF supaya HRD hanya perlu membuka dan mengunduh satu file, bukan tiga lampiran terpisah.

Pengajuan pinjaman atau KPR

Pengajuan kredit ke bank biasanya butuh slip gaji, rekening koran, dan KTP sebagai satu berkas pendukung. Menggabungkannya menghindari masalah kalau sistem hanya mengizinkan satu file per kolom.

Hasil scan beberapa dokumen terpisah

Kalau Anda scan beberapa halaman atau dokumen secara terpisah dan hasilnya jadi beberapa file PDF, gabungkan kembali jadi satu dokumen untuk diarsipkan atau dikirim — jauh lebih cepat dibanding scan ulang semuanya sekaligus.

Cara Gabungkan PDF (Langkah demi Langkah)

1

Buka alat Gabungkan ShrinkPDF

Kunjungi ShrinkPDF.fyi/merge.html menggunakan Chrome, Safari, atau Firefox. Tidak perlu instal aplikasi, tidak perlu daftar akun.

2

Unggah semua file PDF Anda

Klik "Pilih File PDF" dan pilih beberapa file sekaligus, atau tambahkan satu per satu dengan menyeret file ke kotak unggah.

3

Atur urutan penggabungan

Seret setiap file ke posisi yang Anda inginkan. Urutan file di layar akan menjadi urutan halaman di dokumen hasil gabungan.

4

Unduh file gabungan

Klik tombol unduh untuk menyimpan satu file PDF berisi semua dokumen yang tadi digabungkan. Semua proses terjadi di browser Anda, file tidak dikirim ke internet.

🔒 Privasi Kamu Terjaga

ShrinkPDF memproses semua file langsung di browser Anda menggunakan JavaScript. KTP, slip gaji, transkrip nilai, semua tidak pernah meninggalkan perangkat Anda dan tidak dikirim ke server mana pun.

Apakah Kualitas Berkurang Saat Digabungkan?

Menggabungkan PDF itu tanpa kehilangan kualitas

Menggabungkan PDF berbeda dari mengompres. Alih-alih merender ulang setiap halaman jadi gambar, alat Gabungkan menyalin isi asli setiap halaman — teks, font, gambar, grafik vektor — langsung ke file gabungan, persis seperti aslinya. Tidak ada yang di-resample, dikompres ulang, atau dirasterisasi. Ukuran file gabungan kira-kira sama dengan jumlah ukuran semua file asli.

Artinya: teks tetap bisa diseleksi dan dicari, gambar tetap dalam resolusi aslinya, dan tidak ada perbedaan kualitas yang terlihat antara halaman di file gabungan dengan halaman yang sama di file aslinya.

Mengatur Urutan File yang Benar

Urutan Anda mengunggah file belum tentu urutan yang Anda inginkan di hasil akhir — mudah lupa urutannya kalau sudah menambahkan lima atau enam file. Beberapa cara supaya lebih rapi:

  • Beri nama file dengan awalan angka sebelum diunggah (1-transkrip.pdf, 2-suratrekomendasi.pdf) — supaya lebih mudah dikenali saat mengatur urutan.
  • Seret satu file dalam satu waktu daripada mencoba mengatur beberapa sekaligus, terutama di HP di mana area seret lebih kecil.
  • Periksa jumlah halaman setiap file setelah diunggah — kalau ada file yang menunjukkan 1 halaman padahal seharusnya 3, kemungkinan Anda memilih file yang salah atau file yang rusak.

Gabungkan vs Kompres — Sering Butuh Dua-duanya

Menggabungkan dan mengompres menyelesaikan masalah yang berbeda, dan banyak kebutuhan nyata butuh keduanya: gabungkan beberapa dokumen jadi satu file, lalu kompres file gabungan itu supaya muat batas ukuran sistem pendaftaran. Karena menggabungkan tidak mengurangi kualitas, file gabungan bisa jadi lebih besar dari batas yang diizinkan meskipun setiap file aslinya sudah cukup kecil.

Kalau file gabungan Anda masih terlalu besar setelah digabungkan, lanjutkan dengan alat Kompres ShrinkPDF — lihat panduan lengkap cara kompres PDF untuk memilih tingkat kompresi yang tepat.

✓ Gabungkan PDF Anda Sekarang — Gratis

Tanpa daftar. Tanpa batas jumlah file. File Anda tidak meninggalkan perangkat Anda.

Pertanyaan yang Sering Diajukan

Apakah kualitas file berkurang setelah digabungkan? ▼
Tidak. Menggabungkan PDF bekerja di level struktur file — setiap halaman disalin apa adanya ke file baru tanpa dirender ulang atau dikompres. Teks tetap bisa diseleksi dan dicari, gambar tetap dalam resolusi aslinya.
Apakah ada batas jumlah file yang bisa digabungkan? ▼
Tidak ada batas jumlah file atau ukuran total, karena semua proses berjalan langsung di browser Anda, bukan diunggah ke server yang punya batasan penyimpanan.
Apakah perlu memasukkan email untuk menggabungkan PDF? ▼
Tidak. ShrinkPDF tidak memerlukan akun, email, atau pendaftaran untuk alat apa pun, termasuk alat Gabungkan. File hasil gabungan langsung bisa diunduh tanpa syarat tambahan.
Apakah aman menggabungkan dokumen rahasia seperti KTP atau rekening koran? ▼
Ya, aman. ShrinkPDF memproses semua file langsung di perangkat Anda — file tidak pernah diunggah ke server mana pun, termasuk saat proses penggabungan berlangsung.
