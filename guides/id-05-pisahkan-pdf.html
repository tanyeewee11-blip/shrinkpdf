Cara Pisahkan PDF Gratis — Tanpa Upload ke Server

Satu file PDF berisi beberapa dokumen yang tercampur jadi satu, dan Anda cuma butuh sebagian saja? Anda bisa mengambil satu halaman, beberapa halaman, atau memisahkan semuanya jadi file terpisah dalam waktu kurang dari satu menit — gratis, dan file Anda tidak pernah dikirim ke server mana pun.

Kenapa Perlu Memisahkan PDF?

Memisahkan hasil scan campuran

Tumpukan struk, beberapa formulir, atau beberapa dokumen berbeda yang di-scan jadi satu file lebih mudah diberi nama dan diarsipkan satu per satu setelah dipisahkan.

Mengirim hanya yang diminta

Kalau sebuah portal atau HRD cuma butuh satu halaman dari dokumen yang lebih panjang — misalnya halaman foto KTP, atau satu ijazah dari kumpulan transkrip — ambil saja halaman itu tanpa mengirim seluruh dokumen yang tidak relevan.

Menyiasati batas ukuran unggahan per halaman

Kalau satu file PDF yang sangat besar tetap tidak bisa diunggah meskipun sudah dikompres, memisahkannya jadi beberapa file yang lebih kecil (kalau portalnya menerima banyak file) bisa jadi solusi sementara.

Cara Pisahkan PDF (Langkah demi Langkah)

1

Buka alat Pisahkan ShrinkPDF

Kunjungi ShrinkPDF.fyi/split.html. Tidak perlu instal aplikasi, tidak perlu daftar akun.

2

Unggah file PDF Anda

Klik "Pilih File PDF" dan pilih dokumen yang ingin Anda pisahkan.

3

Pilih halaman yang ingin diambil

Pilih satu halaman, beberapa halaman tertentu, rentang halaman (misalnya halaman 5 sampai 10), atau pisahkan setiap halaman menjadi file sendiri-sendiri.

4

Unduh hasilnya

Klik tombol unduh untuk menyimpan file hasil pemisahan. Semua proses terjadi di browser Anda, file tidak pernah dikirim ke internet.

🔒 Privasi Kamu Terjaga

ShrinkPDF memproses semua file langsung di browser Anda. Dokumen apa pun yang Anda pisahkan tidak pernah meninggalkan perangkat Anda.

Apakah Kualitas Berkurang Saat Dipisahkan?

Memisahkan PDF tidak mengurangi kualitas

Sama seperti menggabungkan, memisahkan PDF mengambil setiap halaman persis seperti aslinya dari file sumber — tidak dirender ulang, tidak dikompres ulang. Teks tetap bisa diseleksi dan dicari, gambar tetap dalam resolusi aslinya. Yang berubah hanya halaman mana yang berada di file mana.

Pisahkan vs Kompres — Mana yang Anda Butuhkan?

Kedua alat ini menyelesaikan masalah yang berbeda. Kalau Anda cuma butuh lebih sedikit halaman, pisahkan. Kalau Anda butuh file lebih kecil dengan jumlah halaman yang sama, kompres. Kalau butuh dua-duanya — misalnya dokumen 60 halaman yang cuma butuh 10 halaman, dan 10 halaman itu saja masih terlalu besar untuk diunggah — pisahkan dulu untuk membuang halaman yang tidak perlu, baru kompres sisanya. Lihat panduan lengkap cara kompres PDF untuk memilih tingkat kompresi yang tepat.

✓ Pisahkan PDF Anda Sekarang — Gratis

Tanpa daftar. Tanpa batas halaman. File Anda tidak meninggalkan perangkat Anda.

Pertanyaan yang Sering Diajukan

Apakah kualitas halaman berkurang setelah dipisahkan? ▼
Tidak. Memisahkan PDF mengambil halaman persis seperti aslinya dari file sumber — teks tetap bisa diseleksi, gambar tetap dalam resolusi aslinya. Yang berubah hanya halaman mana yang ada di file mana.
Bisakah saya mengambil hanya satu halaman dari PDF? ▼
Ya. Anda bisa memilih satu halaman, beberapa halaman tertentu, atau rentang halaman (misalnya halaman 3 sampai 7) untuk diekstrak jadi file baru, tanpa perlu memisahkan seluruh dokumen.
Apa bedanya memisahkan dan mengompres PDF? ▼
Memisahkan mengubah halaman mana yang ada di file mana, tapi tidak mengurangi ukuran per halaman. Mengompres mengurangi ukuran file tapi semua halaman tetap dalam satu dokumen.
Apakah ada batas jumlah halaman untuk memisahkan PDF? ▼
Tidak ada batas, karena semua proses berjalan langsung di browser Anda, bukan diunggah ke server yang punya batasan ukuran file.
