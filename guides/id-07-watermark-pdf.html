Cara Menambahkan Watermark ke PDF — Gratis, Tanpa Login

Menandai dokumen dengan RAHASIA, DRAFT, atau SAMPLE biasanya berarti harus mengunduh software tambahan, atau mengunggah file sensitif ke server milik orang lain. Panduan ini menunjukkan cara menambahkan watermark teks ke setiap halaman PDF langsung di browser Anda — tanpa login, tanpa upload — serta cara memilih posisi dan opacity yang tepat agar dokumen tetap mudah dibaca.

Cara Menambahkan Watermark — Langkah demi Langkah

1

Buka Alat Watermark

Kunjungi shrinkpdf.fyi/watermark.html. Tidak perlu daftar, tidak perlu email.

2

Unggah PDF Anda

Klik Choose PDF File atau seret dan lepas. Alat ini membaca jumlah halaman agar Anda tahu watermark akan diterapkan ke seluruh dokumen.

3

Ketik Teks Watermark

Masukkan teks apa pun hingga 60 karakter — "RAHASIA", "DRAFT", "SAMPLE", atau kata-kata Anda sendiri.

4

Atur Posisi, Warna, dan Opacity

Pilih posisi diagonal, tengah, atau bawah, pilih warna, dan sesuaikan opacity agar watermark tidak menutupi konten di bawahnya.

5

Tambahkan Watermark dan Unduh

Klik Add Watermark. PDF hasil akhir langsung dibuat di browser Anda dan langsung terunduh ke perangkat Anda.

Memilih Posisi, Warna, dan Opacity

Opacity lebih penting daripada warna

Faktor paling menentukan apakah watermark terlihat profesional atau justru merusak tampilan dokumen adalah opacity. Sekitar 20-30% membuat teks dan gambar di bawahnya tetap terbaca jelas sementara watermark tetap terlihat. Opacity lebih tinggi (50% ke atas) cocok saat watermark itu sendiri yang ingin ditonjolkan — misalnya menandai dokumen sampel agar tidak digunakan selain untuk ditinjau.

Posisi diagonal yang membentang di tengah halaman adalah pilihan standar untuk watermark seperti "RAHASIA" atau "DRAFT" — lebih sulit dipotong dari screenshot atau salinan cetak dibanding cap kecil di sudut. Watermark di bagian bawah cocok untuk penanda yang lebih halus, seperti nama perusahaan pada penawaran harga, di mana Anda tidak ingin watermark bersaing secara visual dengan konten utama.

Abu-abu adalah warna default paling aman untuk kebanyakan dokumen karena terkesan netral, bukan mendesak. Merah lebih menarik perhatian dan cocok untuk label yang sensitif terhadap waktu seperti "TIDAK BERLAKU" atau "KEDALUWARSA".

Alasan Umum Memberi Watermark pada PDF

Kontrak dan laporan yang masih draft

Tandai dokumen sebagai "DRAFT" selama masih ditinjau, agar tidak ada yang salah mengira versi awal sebagai salinan final yang sudah ditandatangani. Setelah disetujui, beri watermark versi final yang bersih atau kirim file asli tanpa watermark.

Dokumen internal yang rahasia

Beri cap "RAHASIA" pada laporan keuangan, file HR, atau laporan internal sebelum dikirim lewat email, agar jelas secara visual bahwa file tersebut tidak boleh disebarluaskan.

Contoh karya freelancer dan agensi

Beri watermark pada contoh portofolio, mockup, atau penawaran harga dengan nama bisnis atau "SAMPLE" agar karya draft tidak digunakan tanpa pembayaran.

Watermark, Lalu Kompres atau Beri Nomor Halaman

Menambahkan watermark hanya menambah sedikit konten ke halaman, jadi ukuran file hampir tidak berubah — ini bukan pengganti kompresi. Jika file yang sudah diberi watermark masih perlu masuk batas ukuran unggah suatu portal, jalankan lewat alat Compress ShrinkPDF setelah watermark (lihat panduan lengkap kompres PDF).

Jika dokumen juga perlu nomor halaman — umum untuk kontrak dan laporan — gunakan alat Add Page Numbers sebelum atau sesudah watermark; keduanya tidak akan saling mengganggu karena digambar di posisi berbeda pada halaman.

Pertanyaan yang Sering Diajukan

Bisakah watermark dihapus setelah ditambahkan? ▼
Tidak. Setelah diterapkan, watermark menjadi bagian permanen dari halaman dan tidak bisa dihapus lagi lewat alat ini. Simpan file asli tanpa watermark jika sewaktu-waktu Anda membutuhkannya kembali.
Apakah watermark akan mengganggu keterbacaan dokumen? ▼
Tidak, selama opacity-nya diturunkan. Pengaturan sekitar 20-30% membuat watermark tetap terlihat tanpa menutupi teks atau gambar di bawahnya. Opacity lebih tinggi cocok saat Anda ingin watermark itu sendiri yang menonjol.
Apakah watermark otomatis diterapkan ke semua halaman? ▼
Ya. Teks, posisi, warna, dan opacity yang sama diterapkan secara konsisten ke setiap halaman PDF dalam satu proses — tidak perlu mengulang untuk tiap halaman.
Bisakah saya pakai teks sendiri selain "RAHASIA"? ▼
Ya. Kolom teks menerima hingga 60 karakter apa pun — nama perusahaan Anda, "DRAFT", "SAMPLE", "SALINAN", atau pesan khusus Anda sendiri.
Amankah memberi watermark pada dokumen rahasia? ▼
Aman. Proses watermark berlangsung sepenuhnya di browser Anda menggunakan JavaScript — file tidak pernah diunggah ke server mana pun, yang penting untuk dokumen sensitif seperti kontrak, laporan keuangan, dan hasil scan KTP.
