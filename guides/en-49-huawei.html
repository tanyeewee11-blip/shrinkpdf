Compatibility — Which Huawei Phones This Works On

ShrinkPDF is a browser-based tool that works on any phone with a modern browser. For Huawei phones specifically:

Compatibility by device type
  • Huawei phones with Google Play (older models, pre-2019): Full compatibility. Use Chrome, Firefox, or any browser.
  • Huawei phones with HarmonyOS (P40, Mate 40, P50, Mate 50, and newer): Full compatibility via Huawei Browser or any browser installed from AppGallery.
  • Huawei phones with EMUI without Google Play (P30 and later in some markets): Full compatibility via Huawei Browser.
  • Huawei tablets (MatePad series): Full compatibility — larger screen makes the interface more comfortable to use.

The key point: ShrinkPDF does not require any app download from Google Play or AppGallery. It runs as a website in any browser. The only requirement is a modern browser — Huawei Browser (built-in on all Huawei devices) works perfectly.

Method 1: Using Huawei Browser (Built-In)

Huawei Browser is pre-installed on all Huawei phones and is based on Chromium, which means it supports all the web technologies ShrinkPDF uses.

1

Open Huawei Browser

Tap the Huawei Browser icon on your home screen or in the app drawer. It looks like a blue compass or globe icon.

2

Go to ShrinkPDF

Tap the address bar at the top and type shrinkpdf.fyi. Tap Go or the Enter key. The full ShrinkPDF tool loads — the interface is mobile-optimised and works well on phone screens.

3

Upload Your PDF

Tap Choose PDF File. Huawei's file picker (Files app) opens. Navigate to where your PDF is stored — typically in Internal Storage → Downloads, or in the Huawei Files app under "Documents". Select your PDF.

4

Choose Compression Level and Compress

Tap Balanced for general use, or Maximum for government portal uploads with strict size limits. Tap Compress PDF Now. Wait a few seconds for processing to complete.

5

Download the Result

Tap Download. Huawei Browser saves the compressed PDF to Internal Storage → Downloads. A notification appears at the top of the screen — tap it to open the file, or open the Huawei Files app to find it in Downloads.

Method 2: Using Chrome or Other Browsers

If you have Chrome, Firefox, or Opera installed on your Huawei phone (available from AppGallery or sideloaded), you can use any of these. The steps are identical — go to shrinkpdf.fyi, upload, compress, download. The choice of browser doesn't affect how ShrinkPDF works.

For Huawei phones without Google Play: Chrome is available for manual download as an APK from various sources, but Huawei Browser is simpler and equally capable for this purpose. There is no need to install Chrome specifically for ShrinkPDF.

How to Find Your PDF on a Huawei Phone

The most common question when using ShrinkPDF on Huawei phones is finding the PDF after tapping "Choose PDF File." Here's where PDFs typically live on Huawei devices:

  • Downloads folder: PDFs downloaded from browsers or email attachments go here. Path: Internal Storage → Downloads.
  • Huawei Files app: Open Files → tap "Documents" category — it shows all PDF files on your device regardless of where they're stored.
  • WhatsApp: PDFs received via WhatsApp are in Internal Storage → WhatsApp → Media → WhatsApp Documents.
  • Email attachments: Save the attachment first by tapping the download icon in your email app, then find it in Downloads.
  • Cloud storage: If your PDF is in Huawei Cloud, Google Drive, or OneDrive, the file picker may show cloud locations. You can also download the PDF to local storage first and then upload to ShrinkPDF.

Compressing a PDF Received via WhatsApp or Email

If someone sends you a PDF via WhatsApp or email that you need to compress before forwarding to a portal:

  1. Save the file first: In WhatsApp, long-press the PDF message → tap the download icon to save to local storage. In email, tap the attachment → tap Save.
  2. Find it in Downloads: Open Huawei Files → Downloads and confirm the file is there.
  3. Open ShrinkPDF in Huawei Browser and tap Choose PDF File.
  4. Navigate to Downloads and select the file.
  5. Compress and download. The compressed version saves to Downloads alongside the original.
Tip: rename the file after compression

ShrinkPDF downloads the compressed file with the same name as the original. If you need to keep both versions, rename the compressed file immediately after downloading — otherwise it's easy to mix them up. In the Huawei Files app, long-press the file → Rename.

Frequently Asked Questions

Does ShrinkPDF work on Huawei phones without Google Play? ▼
Yes — ShrinkPDF is a website, not an app. It runs in Huawei Browser (pre-installed on all Huawei devices) without requiring any Google Play installation. Huawei Browser is Chromium-based and fully supports all the web technologies ShrinkPDF uses, including the File API for local file processing.
Where does the compressed PDF save on a Huawei phone? ▼
The compressed PDF saves to Internal Storage → Downloads by default when using Huawei Browser. You can find it in the Huawei Files app under Downloads or under the Documents category. A download notification also appears at the top of the screen after tapping Download.
Is ShrinkPDF available in AppGallery? ▼
No — ShrinkPDF is not an app in AppGallery. It's a website that runs in your browser. Open Huawei Browser and go to shrinkpdf.fyi — that's all you need. No installation required.
Can I compress PDFs on a Huawei MatePad tablet? ▼
Yes — ShrinkPDF works on Huawei MatePad tablets the same way it works on phones. The larger screen makes the interface more comfortable. Open Huawei Browser, go to shrinkpdf.fyi, and the process is identical.
Does the compressed PDF stay private on a Huawei phone? ▼
Yes. ShrinkPDF processes files entirely in your browser — your PDF never leaves your device and is not sent to any server. This applies regardless of which phone or browser you use. For sensitive documents like payslips, IC copies, or bank statements, this is particularly important.
My Huawei phone says the file format is not supported. What should I do? ▼
This can happen if the PDF is corrupted, password-protected, or in a non-standard format. Try opening the PDF first in Huawei's built-in document viewer to confirm it opens correctly. If it's password-protected, remove the password first. If it opens fine elsewhere but ShrinkPDF can't read it, the PDF may use unusual encoding — try a different browser on the same phone.
