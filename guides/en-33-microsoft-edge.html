How to Compress a PDF in Microsoft Edge

Microsoft Edge is the default browser on Windows 10 and 11 and has a built-in PDF viewer — but it can't compress PDFs. For compression, ShrinkPDF works seamlessly in Edge without any extensions, add-ons, or installations. If Edge is already open, you're one tab away from compressing any PDF.

Step-by-Step: Compress PDF in Edge

1

Open a New Tab in Edge

Open Microsoft Edge and go to shrinkpdf.fyi in a new tab. You can type it directly into the address bar.

2

Click "Choose PDF File"

Click the upload button. Windows File Explorer will open. Navigate to your PDF — check Downloads, Documents, or Desktop — and select it.

3

Choose Compression Level

Select Balanced for most documents. Select Maximum for scanned files, forms, or ICs where you need the smallest possible file size.

4

Download the Compressed PDF

Click Download. Edge shows a download notification at the top right of the browser. The file saves to your Downloads folder. Click Open file in the notification to preview it immediately in Edge's built-in PDF viewer.

Edge tip: preview before sharing

After downloading, Edge automatically offers to open the compressed PDF in its built-in viewer. Use this to quickly check text is sharp and layout is intact before you email or upload the file.

Compressing a PDF Already Open in Edge

If you've opened a PDF directly in Edge's viewer and want to compress it:

  1. With the PDF open in Edge, press Ctrl+S (or right-click → Save as) to save it to your computer first
  2. Open a new tab and go to shrinkpdf.fyi
  3. Upload the saved PDF, compress it, and download the result

Edge's PDF viewer doesn't have a "Save compressed" option — you need to save the original first, then compress separately.

What Edge's Built-In PDF Tools Can and Can't Do

Edge has a capable built-in PDF viewer with several features — but compression isn't one of them:

  • Can do: View, scroll, zoom, search text, rotate pages, highlight text, draw/annotate, fill forms, add comments, print
  • Cannot do: Compress or reduce file size, merge multiple PDFs, split pages, convert to other formats

For anything beyond viewing and basic annotation, a dedicated tool like ShrinkPDF handles what Edge's built-in viewer can't.

Frequently Asked Questions

Can Microsoft Edge compress PDFs on its own? ▼
No. Edge's built-in PDF viewer can view, annotate, and print PDFs, but it has no compression feature. To compress a PDF in Edge, open ShrinkPDF in a new tab — it runs entirely in the browser without any extensions or add-ons needed.
Do I need to install an Edge extension to compress PDFs? ▼
No. ShrinkPDF is a website that runs in any browser tab, including Edge. There are no extensions to install. Open shrinkpdf.fyi in Edge, upload your PDF, compress it, and download — that's the entire process.
Where does the compressed PDF download to in Edge? ▼
By default, Edge saves downloads to your Downloads folder (C:\Users\[username]\Downloads). A notification appears at the top right of Edge when the download completes. You can click "Open file" to preview it immediately in Edge, or "Show in folder" to locate it in File Explorer.
Does ShrinkPDF work on Edge for Mac and Android? ▼
Yes. ShrinkPDF works in Edge on Windows, Mac, Android, and iOS. The process is identical — go to shrinkpdf.fyi in Edge, upload your PDF, compress, and download. On mobile Edge, the file saves to your Downloads folder on Android or Files app on iOS.
