Choosing Position, Format and Starting Number

Bottom centre is the safe default

For reports, theses, and formal submissions, bottom-centre numbering is the convention readers expect. Bottom-right is a common alternative that can be easier to spot when quickly flipping through a printed stack of pages. Top-centre suits documents where the bottom margin is already used for a footer or signature block.

The format you choose depends on how the document will be used. Plain numbers ("1", "2", "3") suit most documents. "Page 1 of 10" is useful when readers need to know how many pages remain — common for contracts, so a reader can confirm no pages are missing from a printed or scanned copy.

If your document has a cover page or title page that shouldn't be numbered, use the skip option to start numbering from the second page onward while keeping the starting number at 1 for that first counted page.

Common Reasons to Number PDF Pages

Academic reports and theses

University and school submissions are typically expected to have consistent page numbering, often starting after a cover page or table of contents.

Contracts and agreements

Numbered pages make it easy to reference a specific clause during discussion or signing, and to confirm no pages are missing from a printed copy before signatures are collected.

Printed handouts

When a multi-page document is printed and physically handed out, page numbers keep pages in the right order if they get shuffled or dropped.

Numbering Alongside Watermarks or Compression

Page numbers and watermarks are drawn in different areas of the page, so applying both to the same document doesn't cause any conflict. If you also need to mark a document DRAFT or CONFIDENTIAL, use the Add Watermark tool either before or after numbering.

Frequently Asked Questions

Can I start numbering from a page other than 1? ▼
Yes. Set the starting number field to any value, useful when a document continues the numbering of a previous file, such as an appendix.
Can I skip numbering on a cover page? ▼
Yes. The tool lets you skip a set number of pages at the start of the document so a cover page or title page stays unnumbered while the rest of the document numbers normally.
Where should page numbers go on a formal document? ▼
Bottom centre is the most common and universally expected placement for reports, theses, and contracts. Bottom right is also common and can be easier to spot when flipping through a printed stack.
What does the "1 of 10" format mean? ▼
It shows both the current page number and the total page count, such as "3 / 10", which helps readers know how many pages remain in the document.
Is it safe to number confidential documents? ▼
Yes. Page numbers are drawn onto the PDF entirely inside your browser using JavaScript — the file is never uploaded to a server at any point.
