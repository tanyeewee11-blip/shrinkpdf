How to Split a PDF — Extract Pages or Divide Into Separate Files

Sometimes you don't need a whole PDF — just one page out of a 40-page scan, or a long document broken into smaller files so each part can be filed, renamed, or submitted on its own. Splitting handles both of these, and it's a different operation from compressing: this guide covers how splitting works, when to use it instead of (or alongside) compression, and how to pull out exactly the pages you need.

Does Splitting Affect Quality?

Splitting is lossless

Like merging, splitting works at the file-structure level rather than re-rendering anything. Each extracted page is copied directly from the source document — same text, same embedded images at the same resolution, same formatting. Nothing is resampled or recompressed in the process. A page extracted from a 50MB scanned PDF will be roughly 1/50th of that file's size (assuming similar content per page), not because anything was compressed, but because you're now only keeping a fraction of the original pages.

This means splitting alone won't help if your problem is that individual pages are too large — for that, you need to compress (see the comparison section below).

Extract One Page, a Range, or Every Page

  • Single page: useful for pulling out just your IC photo page from a longer scanned document, or grabbing page 12 of a contract that someone specifically asked for.
  • Page range: useful for extracting a specific chapter, section, or set of related pages (e.g. pages 5–10) without including the rest of the document.
  • Every page as a separate file: useful when a multi-page scan actually contains several unrelated documents that were scanned together and need to be filed individually.
Not sure which pages you need?

Open the PDF in your browser or a PDF viewer first and note down the page numbers before starting the split — it's much faster than guessing and re-splitting if you get it wrong the first time.

Common Reasons to Split a PDF

Separating mixed scans

A stack of receipts, several forms, or multiple invoices scanned together as one file are easier to file, rename, or submit individually once split apart.

Submitting only what's asked for

If a portal or recruiter only needs page 2 of a longer document (an ID photo page, a single certificate from a bound transcript), extracting just that page avoids sending an entire unrelated document along with it.

Working around upload size limits page by page

If a single very large PDF won't upload to a portal even after compressing, splitting it into smaller chunks (if the portal accepts multiple files) can be a practical workaround.

Split vs Compress — Which Do You Need?

These solve different problems:

Reduces total file sizeNo (per page, content is unchanged)Yes
Changes how many pages are in a fileYesNo
Keeps text searchable and images sharpYes — unchanged from originalVisually close, but text becomes part of a rasterized image

If you only need fewer pages, split. If you need a smaller file with the same pages, compress. If you need both — say, a 60-page scan where you only need 10 pages, and even those 10 pages add up to too many MB for a portal — split first to remove what you don't need, then compress what's left.

Frequently Asked Questions

Does splitting a PDF reduce the quality of the pages? ▼
No. Splitting extracts pages exactly as they are in the source file — text stays selectable, images keep their original resolution, and nothing is re-rendered or recompressed. The only thing that changes is which pages are in which file.
Can I extract just one page from a PDF? ▼
Yes. You can select a single page, several individual pages, or a page range (e.g. pages 3 to 7) to extract as a new file, without needing to split the entire document.
What's the difference between splitting and compressing a PDF? ▼
Splitting changes which pages are in which file but doesn't reduce file size per page. Compressing reduces file size but keeps all pages in one document. If a file is both too large and contains pages you don't need to send together, split first to remove unnecessary pages, then compress what's left.
Is there a page limit for splitting a PDF? ▼
No. ShrinkPDF's split tool works on PDFs of any length, since processing happens locally in your browser rather than on a server with file size restrictions.
Can I split a PDF on my phone? ▼
Yes. The split tool works the same way in any mobile browser as on desktop — upload, choose pages, and download the result directly to your phone.
Is it safe to split confidential documents? ▼
Yes. ShrinkPDF processes everything locally in your browser — files are never uploaded to any server during splitting, which matters if you're pulling a single ID page or financial document out of a larger scan.
