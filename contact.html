Before you write in — quick answers
Q
My compressed PDF has no selectable text — is this a bug?
No, this is expected. Compression works by re-rendering each page as an image. Text becomes part of the image and is no longer selectable. Keep your original file if you need the text layer.
Q
The tool isn't working — nothing happens when I click Compress.
Try refreshing the page and using a different browser. Chrome and Edge work best. If the issue persists, tell us your browser version and operating system and we'll look into it.
Q
Can I use ShrinkPDF for my business or clients?
Yes, no restrictions. The tools are free for personal and commercial use.

What This Site Is Not

ShrinkPDF is not an AI-generated content farm, not a link-building scheme, and not a white-label product. Every guide on this site was written by a human who has actually used the tool and understands the use case it describes. The site exists because the problem it solves — reducing PDF file sizes privately, without uploading to a server — is one that affects real people every day, and the existing solutions all come with catches.

If you're a journalist, researcher, or publisher who wants to verify anything about how the tool works, what data we collect, or how the site is run, we're happy to answer directly. The technology is open-source and the privacy claims are technically verifiable in any browser's DevTools.
