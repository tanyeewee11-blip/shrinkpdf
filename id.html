Pertanyaan yang Sering Diajukan
Apakah kompresi mempengaruhi kualitas PDF saya? ▼
Untuk PDF berisi teks (kontrak, laporan, formulir), kualitas hampir identik setelah kompresi. Untuk PDF bergambar, mode ringan atau sedang tidak akan terlihat bedanya. Hanya mode maksimum yang mungkin sedikit mengurangi ketajaman gambar.
Apakah file saya aman? Apakah Anda menyimpan PDF saya? ▼
File Anda diproses 100% di browser menggunakan JavaScript. File tidak pernah diunggah ke server mana pun. Kami tidak bisa melihat, mengakses, atau menyimpan file Anda.
Saya perlu upload PDF ke SSCASN / portal pemerintah tapi file terlalu besar, bisa pakai ini? ▼
Ya! Kebanyakan portal pemerintah dan universitas membutuhkan PDF di bawah 1–2MB. Gunakan mode "Maksimum" untuk mendapatkan file sekecil mungkin untuk diunggah.
Berapa ukuran file maksimum yang bisa dikompres? ▼
Tidak ada batas dari pihak kami. File yang sangat besar (100MB ke atas) mungkin butuh beberapa detik ekstra tergantung kemampuan perangkat Anda, karena semua pemrosesan dilakukan secara lokal di browser.
