Frequently Asked Questions

Is there a limit to how many PDFs I can merge?
No. You can merge as many PDF files as you need. There is no file count limit or file size limit on ShrinkPDF.
Are my files uploaded to a server?
No. All merging happens entirely in your browser using JavaScript. Your files are never sent to any server.
Can I reorder pages before merging?
Yes. After uploading, drag the files into the order you want before clicking Merge.
What if my merged PDF is too large to upload somewhere?
Use the Compress tool after merging to reduce the file size without losing quality.
